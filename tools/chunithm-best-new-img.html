新しい算出方法により、より正確なベスト枠・新曲枠を提供できるようになりました。

CHUNITHM-NET上での表示順を推測してその順序に並べています。

使い方

下記『chunirec 情報の登録』にchunirecのユーザーネームを入力します。

スマートフォンの場合、ユーザーネームは以下の画像の下線部にあります。

APIアクセストークンについて

chunirec APIアクセストークンを登録することで、以下のようなメリットがあります。

  • 非公開アカウントであっても、自分のアカウント情報を取得できます。
  • アクセス制限にかかりにくくなります。

ただし、以下の点に注意してください。

  • APIアクセストークンは第三者に知られないようにしてください。
  • 他人の非公開アカウント情報を取得することはできません。
  • アクセストークンを再発行した場合、古いアクセストークンは無効になります。

本機能は、多くのアクセス制限が発生している現状への緊急避難的な措置であり、将来的に変更または廃止される可能性があります。
本機能に関するお問い合わせに関して積極的なサポートを行うことはありませんので、ご自分で対応できる方のみご利用ください。

APIアクセストークンの登録はこちら

chunirec API アクセストークン登録

chunirec API アクセストークンはchunirec 開発者向けサイトにログインしたあと、アカウント管理画面から発行できます。

使用しない場合は必ず空欄にしてください。

注意事項

  • ユーザー名が入力されていない場合、本ツールは動作しません。
  • chunirec側で非公開アカウントに設定されている場合、エラーが発生します。
  • chunirecがメンテナンスを行っているなどの事情で、本ツールが使用できないことがあります。
  • 本ツールで取り扱う『達成日』『更新日』は、厳密にはスコアがchunirecに登録された日を指します。
    実際にそのスコアを達成した日とは異なる可能性があることに留意してください。
  • chunirec上で表示されていない楽曲に関しては、計算結果が正しいことを保証できません。

不具合

  • 『プレビューを表示する』にチェックが入っていなくても一時的にプレビューと同じ描画領域が表示されます。
    一時的に表示が崩れることがありますが、使用する上での問題はありません。
  • ScoreSupにデータが登録されていた場合、一部楽曲が重複することがあります。
    ScoreSupに一回アクセスすることでデータが正常に反映されます。

設定

chunirec 情報の登録

chunirec ユーザーネーム

その他の設定

chunirecに反映されていない新曲のスコアは、ScoreSupで入力することができます。

ScoreSupを利用している方で、楽曲が重複しているバグが起こっている場合は、一度ScoreSupを開いてからこのツールを利用すると修正されます。
